### Django Template Language(DTL)
 - Django Template에서 사용하는 문법
   - (주의) python이 동작하는 게 아님
 - **DTL Language**
   - 변수 (Variabel)
    `{{variable}}`
        view의 context로 넘긴 데이터를 접근할 수 있음
   - 필터 (Filters)
    `{{variable|filter}}`
   - 태그 (Tags)
    `{% tag %}`
        - 반복 또는 논리를 수행하여 제어 흐름을 만들거나 특수한 기능을 수행
   - 주석 (Comments)
    `{# 한줄주석 #}`

        ```
        {% comment %}
        여러줄
        주석
        {% endcomment %}
        ```
